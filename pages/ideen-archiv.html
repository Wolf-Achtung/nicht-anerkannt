Deine Idee einreichen

Das Ideen-Archiv ist ein offenes System. Du kannst Denkproben, Widerspruchssalon-Themen oder eigene Beiträge über einen Pull Request auf GitHub einreichen.

So funktioniert es:

  1. Forke das Repository auf GitHub.
  2. Erstelle eine neue Datei im Ordner data/ mit deiner Idee.
  3. Beschreibe deine Denkprobe, dein Thema oder deinen Widerspruch.
  4. Öffne einen Pull Request mit einer kurzen Begründung.
