Submit Your Idea

The Ideas Archive is an open system. You can submit thinking challenges, Contradiction Salon topics, or your own contributions via a pull request on GitHub.

How it works:

  1. Fork the repository on GitHub.
  2. Create a new file in the data/ folder with your idea.
  3. Describe your thinking challenge, topic, or contradiction.
  4. Open a pull request with a brief explanation.
